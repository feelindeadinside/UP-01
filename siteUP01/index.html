Техническое задание

1. Введение

1.1 Наименование системы: Система управления фармакологической компанией
1.2 Основание для разработки: Анализ предметной области управления фармакологической компанией

2. Цель разработки

Целью разработки данной системы является создание автоматизированного решения для управления компанией по продажам фармацевтических средств. Это позволит улучшить организацию рабочего процесса, повысить эффективность использования продаж и контроля товаров.

3. Область применения

Система будет использоваться для управления количеств продаж компании. Это способствует более эффективному управлению бизнесом и оптимизации процессов в организации.

4. Требования к системе

4.1 Функциональные требования:
1. Авторизация сотрудников по их роли
2. Роль сотрудника(создание, чтение, редактирование и удаление данных).
3. Роль клиента (просмотр всех лекарств и их цены, создание продаж и просмотр продаж).
4.2 Нефункциональные требования
1. Интерфейс должен быть интуитивно понятным и удобным для пользователей
2. Система должна обеспечивать защиту данных и безопасную аутентификацию пользователей.
3. Высокая скорость обработки запросов к базе данных.

5. Нефункциональные требования

5.1 Защита данных: система должна обеспечивать резервное копирование данных и защиту от несанкционированного доступа.
5.2 Отказоустойчивость: система должна корректно обрабатывать ошибки и не прерывать работу при сбоях.

6. Условия эксплуатации

Система должна работать на операционной системе Windows и быть доступной через приложение, разработанное на C#, с использованием ASP.NET для графического интерфейса и SQLite для управления базой данных.

7. Требования к программному обеспечению

7.1 Язык программирования: C#
7.2 Среда разработки: Rider
7.3 Среда выполнения: Windows 10
7.4 База данных: SQLite

8. Требования к документированию

К системе должны быть предоставлены следующие документы:
Руководство пользователя
Руководство администратора
Техническое руководство по установке и настройке системы

10.1 Приемка: система будет считаться принятой, если она соответствует всем требованиям ТЗ и успешно проходит все этапы тестирования.
10.2 Контроль качества: регулярная проверка и тестирование системы перед внедрением.

Руководство пользователя

1. Общие сведения

1.1 Назначение
Приложение «Каталог продукции фармакологической компании» предназначено для сотрудников и клиентов компании, желающих получить актуальную информацию о продукции, сети аптек, сотрудниках и партнерах. С помощью приложения можно:
Просматривать информацию о продукции: найти нужный препарат, узнать его название, состав, показания и противопоказания, а также дату выпуска и срок годности.
Узнавать о сети аптек: просмотреть информацию об аптеках, таких как адрес, контактный номер телефона и график работы.
Получать сведения о сотрудниках: просматривать имена и должности сотрудников компании, а также контактную информацию при наличии.
Изучать информацию о партнерах: узнать названия партнерских организаций, страну их расположения и другие контактные данные.
Ознакомиться с отзывами клиентов: просматривать отзывы о продукции с именем автора, оценкой и датой отзыва.
1.2 Требования к пользователю
Для использования приложения требуется:
Смартфон или планшет с операционной системой Android.
Доступ в интернет.

2. Описание приложения

2.1 Основные функции
Приложение «Каталог продукции фармакологической компании» предоставляет следующие функции для просмотра информации: подробные описания продукции, аптек, сотрудников, партнеров и отзывов клиентов.
2.2 Интерфейс
Приложение имеет простой и интуитивно понятный интерфейс. Основной экран приложения содержит список категорий: Продукция, Аптеки, Сотрудники, Партнеры и Отзывы.

3. Инструкция по эксплуатации

3.1 Запуск приложения
Для запуска приложения необходимо открыть его на вашем устройстве.
3.2 Выбор категории
На главном экране приложения выберите категорию, информацию о которой вы хотите получить.
3.3 Просмотр информации
После выбора категории вам будет представлен список элементов этой категории. Выберите интересующий элемент для просмотра подробной информации о нем.
